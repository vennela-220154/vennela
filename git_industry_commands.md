
# Git Industry Commands

## 1. Git Configuration Commands

### Command: git config --global user.name
Syntax: git config --global user.name "Your Name"

Purpose: Git commits lo username set cheyadaniki.

Example:
git config --global user.name "Vennela"


### Command: git config --global user.email
Syntax: git config --global user.email "email@example.com"

Purpose: Git commits ki email set cheyadaniki.

Example:
git config --global user.email "vennela@gmail.com"


### Command: git config --list
Syntax: git config --list

Purpose: Git configuration settings list chupisthundi.

Example:
git config --list


### Command: git config --unset
Syntax: git config --global --unset user.name

Purpose: Already set chesina configuration remove cheyadaniki.

Example:
git config --global --unset user.name
